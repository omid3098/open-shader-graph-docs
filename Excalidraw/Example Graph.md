Traverse Rules:
- Start from the main shader
- Start from output nodes
    - each output node will generate a code like 'ALBEDO = {{Color}}" line in the expected shader. if it has multiple lines like for Normal, Roughness, etc.. each one of these lines need to refer to a pin.
    like 'ROUGHNESS={{Roughness}} which {{Roughness}} is the pin name.
    - {{Color}} should be replaced by the value of an input that matches the name {{Color}} which in this case is the only input pin and its value is '1/out'
    - So before generating the node code from template, first we need to  find out what is the code for each pin. in this case, what does '1/out' mean? and then replace them all into the node code. ^IC7FXTW3
